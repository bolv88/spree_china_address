SpreeChinaAddress
=================

Spree checkout address for china.

Add city, city-area and reciver name to address model.

Remove firstname, lastname.

how to use in spree
=======

- cd your_store
- bundle exec rake spree_china_address:install:migrations
- bundle exec rake db:migrate
